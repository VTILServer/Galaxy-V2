## Galaxy Bot
A very easy to use/setup bot for general purpose usecases like moderation and verification.

# How to setup Galaxy Bot for your server
0) use npm to install required packages, ``npm i``.
1) rename env.example to .env.
2) put your discord bot token in the .env file you created and put your discord user id
3) install pm2 as a global package, ``npm i -g pm2``.
4) start it! ``pm2 start bot.js``.
5) monitor for errors. ``pm2 monit``.
And your done!

# Notes
This bot is built for customizability and ease of use.
